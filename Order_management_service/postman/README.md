# Postman — Order Management Service

## Import

1. Open Postman → **Import** → **Upload Files**.
2. Select **`Order_Management_Service.postman_collection.json`** in this folder.

## Collection variables

| Variable   | Default                 | Purpose |
|-----------|-------------------------|---------|
| `baseUrl` | `http://localhost:8080` | API root |
| `orderId` | *(empty)*               | Set automatically after **Create Order** (201), or paste manually |
